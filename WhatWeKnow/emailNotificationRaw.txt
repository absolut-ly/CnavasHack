** Summary
------------------------------------------------------------

Users seeing slow load times and page errors in Canvas from 2:25 PM MT to=
 3:15 PM MT on 3/3/2021
Overview

 slow load times or page error messages when accessing the site. A code ch=

ng new requests to Canvas to fail. Users were able to access the site agai=
n=2C with expected performance=2C once we had rolled back the deploy and e=
nsured this service was working properly with Canvas application servers.
